SVG Strip Map Generator - User Guide

Welcome to the user guide for the SVG Strip Map Generator. In this guide, we will walk through creating a number of Strip Maps for real world transit networks. Together with the example networks that showcase what this software is capable of, this user guide should provide sufficient guidance and examples for using the software.

Each Unit will cover a different real world example.

  • Unit 1 - Standalone Line (London Underground Jubilee Line)
  • Unit 2 - Branching Line (Copenhagen M1, M2)
  • Unit 3 - Interlining with Common Core, Branches on both ends (UTA TRAX Light Rail)
  • Unit 4 - Loop Lines (Glasgow)

Guides will cover using certain aspects of the software

  • Guide 1 - Station Names
  • Guide 2 - Station Icon Components
  • Guide 3 - Icons
